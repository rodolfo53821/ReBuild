This is a master's degree project.
Create under UFV-DPI master's degree program.

This system defines the best position to construct a dam to supply
a water scarcity in part of the drainage network.
This system uses the RWFlood algorithm to calculate the drainage network.

The grouping files is defined by QtCreator.
In ./ProjetoHidro/docs has a lot of documents about classes in the system.
